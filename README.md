Order and Unordered List Practice
===

<p>Ordered List:
<ol>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
    <li>Fourth item</li>
</ol>
</p>

<p>Indented Ordered List:
<ol>
    <li>First Item</li>
    <li>Second Item</li>
    <li>Third Item</li>
    <ol>
        <li>Indented Item</li>
        <li>Indented Item</li>
    </ol>
    <li>Fourth Item</li>
</ol>
</p>

<p>Unordered List:
<ul>
    <li>First Item</li>
    <li>Second Item</li>
    <li>Third Item</li>
    <li>Fourth Item</li>
</ul>
</p>

<p>Indented Unordered List:
<ul>
    <li>First Item</li>
    <ul>
        <li>Indented Item</li>
        <li>Indented Item</li>
        <li>Indented Item</li>
    </ul>
    <li>Second Item</li>
    <li>Third Item</li>
    <li>Fourth Item</li>
</ul>
</p>

<p>Putting both together:
<ol>
    <li>First Item</li>

    </ul>
    <li>Second Item</li>
    <li>Third Item</li>
    <li>Fourth Item</li>
</ol>
</p>
